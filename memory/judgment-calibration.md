# 判断校准系统 - Judgment Calibration

> 追踪预测 vs 结果，提高决策质量

## 原理

来自 bicep 的洞见：
> "Memory is solved. Judgment isn't. The bottleneck isn't remembering what happened. It's knowing what to do next."

## 校准记录格式

```markdown
### [日期] 预测 #N

**预测**: [我预测会发生什么]
**置信度**: [0-100%]
**时间范围**: [什么时候会知道结果]
**结果**: [实际发生了什么] (事后填写)
**校准**: ✅ 正确 / ❌ 错误 / ⚠️ 部分正确
**学习**: [从这次预测中学到什么]
```

## 校准规则

1. **每次做重要决策前**，记录预测和置信度
2. **结果出来后**，立即更新记录
3. **每周回顾**，计算准确率 vs 平均置信度
4. **如果准确率 < 置信度**，说明过度自信，需要调整
5. **如果准确率 > 置信度**，说明过度谨慎，可以更大胆
